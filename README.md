## E-Commerce Sales Data Analysis
### Project Overview
Analyzed e-commerce sales data (2023â€“2025) to identify revenue trends, top-performing categories, regional performance, and customer purchasing behavior. The project focuses on generating actionable business insights using Microsoft Excel.
### Key Objective
- Identify top-selling categories by revenue
- Analyze month-wise and year-wise sales trends
- Evaluate region-wise performance
### Application Used
MS Excel 2021
### Steps Followed
- Data Cleaning and Transformation - Used IF(),AVERAGEIFS(),DATE(),DATEVALUE(), basic mathematical formulae and power query for handling missing and inconsistent values.
- Data Visualization - generated Pivot table from the cleaned dataset, used charts(Bar, Line and Pie) and trendline for analyzing the growth trend.
- Dashboard Creation - Used all the Charts and filters to create an interactive dashboard.
### Insights Derived
This project helped in understanding sales trends, top-performing categories, and customer behavior. The analysis showed steady sales growth with seasonal fluctuations and identified areas for improvement such as low-performing months and categories. By applying better inventory planning, targeted marketing, and discount strategies, overall business performance can be improved.
